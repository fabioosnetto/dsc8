Tool Usage

  • Click "Add Files" and choose the .dfm and .pas files;
  • Right click the file you want to convert;
  • At this step, all automatic changes was done;
  • Use the "Convert Viewer" to set the non-auto changes (objects order and width);
  • When you want to get the modified/converted files, you can copy from file viewer or right click the file in sidebar and select "Download".
